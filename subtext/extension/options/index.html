What is Subtext?

Subtext is a BS translator for corporate language. It reads investor emails, job postings, LinkedIn posts, and performance reviews — then tells you what they actually mean.

It uses Claude AI to decode the doublespeak, calculate a BS score, and serve up the honest rewrite you deserved in the first place.

Anthropic API Key

Subtext runs on Claude. You'll need an API key from Anthropic. Keys are stored locally in your browser — they never leave your machine.

Don't have a key? Get one at console.anthropic.com →

How it works

  1. Select any text on a webpage — an email, a job posting, a LinkedIn post
  2. Click the ✦ Subtext button that appears
  3. The side panel opens and analyzes the text with Claude
  4. Get a BS score, decoded translations, the honest rewrite, and the reply you actually want to send
